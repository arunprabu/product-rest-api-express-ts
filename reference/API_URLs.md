
# CRUD Operations for Products

## API Endpoints

- **Add Product**
  - Method: `POST`
  - URL: `localhost:3000/api/v1/products`

- **List All Products**
  - Method: `GET`
  - URL: `localhost:3000/api/v1/products`

- **Get Product Details**
  - Method: `GET`
  - URL: `localhost:3000/api/v1/products/:id`
  - Examples:
    - `localhost:3000/api/v1/products/432567`
    - `localhost:3000/api/v1/products/23543`
    - `localhost:3000/api/v1/products/578698`

- **Update Product**
  - Method: `PUT`
  - URL: `localhost:3000/api/v1/products/:id`
  - Example: `localhost:3000/api/v1/products/432567`

- **Delete Product**
  - Method: `DELETE`
  - URL: `localhost:3000/api/v1/products/:id`
  - Example: `localhost:3000/api/v1/products/432567`
